# Customer Churn Prediction System

This project is a simple end-to-end machine learning application that predicts whether a telecom customer is likely to churn. It covers the full flow — from data preprocessing and model training to building a small web app using Streamlit.

## What this project does

The model takes a few customer details like tenure, monthly charges, contract type, etc., and predicts if the customer is likely to leave or stay. It also shows a probability score instead of just a label, which helps understand how confident the model is.

## How it works

- Cleaned and prepared the telecom dataset  
- Created a few useful features (like tenure group, total services, etc.)  
- Converted categorical data into numerical format  
- Trained models like Logistic Regression and Random Forest  
- Compared performance and selected a working model  
- Saved the model and encoders for reuse  
- Built a Streamlit app to take user input and make predictions  

## Tech used

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  
- Git & GitHub
